Create `noise-loop-generator/src/audio/exportEngine.js` with a single exported async function:

```javascript
export async function exportMix({ activeSounds, gains, masterGainValue, durationSec })
```

Parameters:
- `activeSounds`: array of `{ id, type }` objects (from catalog, filtered to currently playing)
- `gains`: `Map<string, number>` — per-sound gain values (from volumeMap)
- `masterGainValue`: number — current master gain (from `getMasterGain().gain.value`)
- `durationSec`: number — 30, 60, or 120

Implementation steps:

**1. Constants and imports:**
```javascript
import { getContext } from './AudioEngine.js';
import audioBufferToWav from 'audiobuffer-to-wav';

const CROSSFADE_SEC = 1.5;
```

**2. Create OfflineAudioContext:**
```javascript
const sampleRate = getContext().sampleRate; // 44100
const totalSec = durationSec + CROSSFADE_SEC;
const totalSamples = Math.ceil(sampleRate * totalSec);
const offlineCtx = new OfflineAudioContext(1, totalSamples, sampleRate);
```

**3. Create offline master gain node:**
```javascript
const offlineMaster = offlineCtx.createGain();
offlineMaster.gain.value = masterGainValue;
offlineMaster.connect(offlineCtx.destination);
```

**4. For each active sound, create per-sound gain + source:**

For noise types (white, pink, brown, grey), generate samples directly into a Float32Array using the exact algorithms from the worklet processors. DO NOT use AudioWorkletNode in OfflineAudioContext — it is unreliable across browsers.

Noise generation functions (private, inside the module):

- `generateWhiteNoise(totalSamples)`: `data[i] = Math.random() * 2 - 1`

- `generatePinkNoise(totalSamples)`: Voss-McCartney filter — initialize `b = [0,0,0,0,0,0,0]`, then for each sample:
  ```
  white = Math.random() * 2 - 1
  b[0] = 0.99886 * b[0] + white * 0.0555179
  b[1] = 0.99332 * b[1] + white * 0.0750759
  b[2] = 0.96900 * b[2] + white * 0.1538520
  b[3] = 0.86650 * b[3] + white * 0.3104856
  b[4] = 0.55000 * b[4] + white * 0.5329522
  b[5] = -0.7616 * b[5] - white * 0.0168980
  data[i] = (b[0]+b[1]+b[2]+b[3]+b[4]+b[5]+b[6] + white*0.5362) / 10
  b[6] = white * 0.115926
  ```

- `generateBrownNoise(totalSamples)`: `lastOut = 0`, then `lastOut = lastOut * 0.998 + white * 0.02; data[i] = lastOut * 3.5`

- Grey noise: generate white noise buffer (same as white), then route through `offlineCtx.createBiquadFilter()` with `type = 'lowshelf'`, `frequency.value = 800`, `gain.value = 10` — matching soundManager.js grey noise filter exactly.

For each noise sound:
```javascript
const noiseBuffer = offlineCtx.createBuffer(1, totalSamples, sampleRate);
const noiseData = noiseBuffer.getChannelData(0);
// Fill noiseData using the appropriate generate function
const src = offlineCtx.createBufferSource();
src.buffer = noiseBuffer;
const gainNode = offlineCtx.createGain();
gainNode.gain.value = gains.get(id);
// For grey: src -> biquadFilter -> gainNode -> offlineMaster
// For others: src -> gainNode -> offlineMaster
gainNode.connect(offlineMaster);
src.start(0);
```

For sample types (rain, wind, thunder), fetch and re-decode:
```javascript
const response = await fetch(`/samples/${id}.wav`);
const arrayBuffer = await response.arrayBuffer();
const audioBuffer = await offlineCtx.decodeAudioData(arrayBuffer);
const src = offlineCtx.createBufferSource();
src.buffer = audioBuffer;
src.loop = true;
const gainNode = offlineCtx.createGain();
gainNode.gain.value = gains.get(id);
gainNode.connect(offlineMaster);
src.connect(gainNode);
src.start(0);
```

**5. Render:**
```javascript
const renderedBuffer = await offlineCtx.startRendering();
```

**6. Apply tail crossfade for gapless looping:**
```javascript
const durationSamples = Math.ceil(sampleRate * durationSec);
const crossfadeSamples = Math.ceil(sampleRate * CROSSFADE_SEC);
const data = renderedBuffer.getChannelData(0);
for (let i = 0; i < crossfadeSamples; i++) {
  const tailFade = 1 - (i / crossfadeSamples); // linear 1 -> 0
  data[i] += data[durationSamples + i] * tailFade;
}
```

CRITICAL: Use `+=` (add fading tail onto head), NOT `=` (replace). The head plays at full amplitude; only the decaying tail energy is blended in.

**7. Create trimmed buffer and encode:**
```javascript
const trimmed = new AudioBuffer({ numberOfChannels: 1, length: durationSamples, sampleRate });
trimmed.copyToChannel(data.slice(0, durationSamples), 0);
const wavArrayBuffer = audioBufferToWav(trimmed); // 16-bit PCM (default)
```

**8. Trigger download:**
```javascript
const blob = new Blob([wavArrayBuffer], { type: 'audio/wav' });
const url = URL.createObjectURL(blob);
const a = document.createElement('a');
a.href = url;
a.download = filename; // passed as param or computed from durationSec
document.body.appendChild(a);
a.click();
document.body.removeChild(a);
setTimeout(() => URL.revokeObjectURL(url), 10000);
```

Use a simple map: `const filenames = { 30: 'noise-mix-30s.wav', 60: 'noise-mix-1min.wav', 120: 'noise-mix-2min.wav' };`
