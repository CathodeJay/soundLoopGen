**renderMixer() function — builds the entire mixer DOM:**

For each catalog entry, create a row `<div>` with these inline styles:
- Row: `display:flex;align-items:center;gap:8px;padding:8px 0;border-bottom:1px solid #2a2a2a;`

Each row contains 3 elements (left to right):

1. **Label** `<span>`: `color:#ffffff;font-size:16px;font-weight:400;min-width:100px;` — textContent from `catalog[i].label`

2. **Toggle button** `<button>`:
   - Inactive state: text `\u25B6 Play`, styles: `background:#1e1e1e;color:#ffffff;border:1px solid #444;font-size:16px;font-weight:600;min-height:44px;padding:4px 12px;border-radius:4px;cursor:pointer;`
   - Active state: text `\u25A0 Stop`, styles: `background:#4caf50;color:#ffffff;border:1px solid #4caf50;` (same size/padding)
   - Click handler logic:
     - If NOT playing: `await startSound(id)` then immediately `setGain(id, volumeMap.get(id))` to override the default gain with the remembered value. Update button to active styles. Enable slider: `slider.disabled = false; slider.style.opacity = '1'; slider.style.pointerEvents = 'auto';` Set `slider.value = volumeMap.get(id)`.
     - If playing: `stopSound(id)`. Update button to inactive styles. Disable slider: `slider.disabled = true; slider.style.opacity = '0.4'; slider.style.pointerEvents = 'none';`
   - Wrap the click handler body in try/catch. On error: `console.error(\`[main] Failed to start ${id}: ${err.message}\`);`

3. **Volume slider** `<input type="range">`:
   - Attributes: `min="0"`, `max="1"`, `step="0.01"`
   - Style: `flex:1`
   - Initial state: `disabled=true`, `opacity:0.4`, `pointerEvents:none`
   - Default value: initialize `volumeMap.set(id, defaultVol)` where `defaultVol` is `DEFAULT_GAIN_NOISE` (0.15) for `type === 'noise'`, `DEFAULT_GAIN_SAMPLE` (0.8) for `type === 'sample'`. Set `slider.value = volumeMap.get(id)`.
   - `input` event listener: `setGain(id, parseFloat(slider.value)); volumeMap.set(id, parseFloat(slider.value));`

**Initial state per row:** Check `isPlaying(id)` — if true, set button to active state and enable slider. (All sounds start inactive, but this handles HMR during dev.)

**After the catalog loop, add:**

4. **Divider** `<div>`: `border-top:1px solid #444;margin:16px 0;`
